# 트레저 웰렛 가이드
다음 가이드는 [트레저 모델 T](https://shop.trezor.io/product/trezor-model-t) 하드웨어 지갑을 비컨 체인과 바이낸스 DEX에서 사용하는 법에 대해 다룹니다.
하드웨어 지갑을 사용하여 암호화폐를 보관할 때 최고의 보안을 유지하세요.

> 현재 트레저 모델 One은 지원되지 않습니다.

요구 사항
트레저에서 비컨 체인과 바이낸스 DEX를 사용하려면:
* 지원 웹 브라우저: 크롬 혹은 파이어폭스
* 브라우저를 통해 비컨 체인 웹 지갑에 대한 접근 (https://binance.org/unlock)
* 최신 버전 의 초기화된 트레저 모델 T 기기
* 컴퓨터에 트레저 브릿지 어플리케이션 설치
